Python 是一種易學、強大的程式語言。

具備如下特點：
1. high-level 資料結構。
2. 簡單有用物件導向編程。
3. 優雅語法。
4. dynamic typing。

Python interpreter 及 standard library 皆以 source 或 binary 型式，存放在 Python 官網上供人免費下載。
官網上同時也提供許多 third party 的 Python modules, programs 及工具。
當然，相關手冊也是有提供的！

官網 http://www.python.org

Python interpreter 具備高度擴充性 (用 C 或 C++)。
對於想製作軟體 extension，Python 也是一個好選擇。

這個 tutorial 介紹了 Python 基本觀念及 Python 功能。
