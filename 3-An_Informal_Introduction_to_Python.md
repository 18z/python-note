3.1.1 數字


直接輸入數學式(數字與加減乘除符號)，Python就可幫忙算囉～

```Python
>>> 2 + 2
4
>>> 50 - 5*6
20
>>> (50 - 5.0*6) / 4
5.0
>>> 8 / 5.0
1.6
```
