在電腦上做了很多事，我們就會想把這些事給「自動化」。
或者是想做一個小型客製化的資料庫。
或是做一個有圖形介面的應用軟體、簡單的遊戲

覺得 C/C++/Java 開發過程中，「撰寫/compile/測試/re-compile」如此開發速度太慢了。
想寫替一個 library 寫 test suite，但發現要寫 testing code 實在很麻煩。 

Python is just the language for you.

你可以寫 Unix shell script 或 Windows batch 來解決問題，但上述解法對於想要開發圖形化應用軟體、遊戲並不適合。
你可以用 C/C++/Java來寫，但你會寫很久拉 XD
相較之下，Python 比較簡單，Windows, Mac OS X 及其他 Unix 作業系統都可以運行，幫助你更快達成你的任務。

千萬不要以為 Python 這麼好用，他就不是一個正規的程式語言。他比 shell scripts 或 batch 檔更厲害。
跟 C 語言比起來，Python 提供更多的錯誤檢查機制。
high-level data types 也是 Python 一大特色，例如：arrarys 跟 dictionaries 的使用。
也因上述優點，Python 跟 Awk, Perl 相比，能處理更大的 problem domain。 

Python 讓我們可以將我們的 program 拆分成 modules ，並讓其他的 Python programs 使用。
這些模組就是 Python 開發的基礎，例如提供 I/O, system calls, sockets 或圖形介面等，提升開發效率。

Python 是一種 interpreted 語言，讓你在開發過程中不必等候 compilation 及 linking。
Interpreter 是可互動式的，讓使用者可以輕易的進行 Python 功能測試或實驗，或者是撰寫 throw-away 的程式等。
他也是一個方便的計算機拉～

Python 可讓使用者寫出簡潔、可讀性高的 code。
用 Python 寫出來的程式通常都比用 C, C++ 或 Java 寫出來的程式要短。原因是：
1. high-level data types 讓開發者可以 single statement 來表達複雜的 operations。
2. 用縮排來代替括號
3. 宣告變數非必要

Python 是 extensible 的：簡單說，有一用 C 寫的程式，想要擴充，你可以用 Python 幫他寫 extension 喔！
Python 之所以叫  Python 是因為 BBC 的一個節目拉 “Monty Python’s Flying Circus”。

學習一種語言，最好的方式就是直接使用它！
接下來我們就來跟 Python interpreter 玩遊戲吧！
